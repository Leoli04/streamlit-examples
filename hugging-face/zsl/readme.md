
## 说明
零样本学习 (Zero-shot learning,简称ZSL) 与传统的机器学习方法不同，因为它涉及“无需”任何训练样本即可识别对象的能力。

[wiki关于ZSL 解释](https://en.wikipedia.org/wiki/Zero-shot_learning)
